## 🚀 Getting Started

### Clone the repository
```bash
git clone https://github.com/praveenkr-07/portfolio.git
```

### Navigate to project directory
```bash
cd portfolio
```

### Open the project
Simply open **index.html** in your browser or use **Live Server** extension in VS Code for live reload:

```bash
# If using Live Server extension
Right-click on index.html → Open with Live Server
```

### Deployment

**Already deployed on:**
- ✅ GitHub Pages: https://praveenkr-07.github.io/portfolio/
- ✅ Vercel: https://portfolio-liart-theta-30.vercel.app/

To deploy to Vercel:
1. Connect your GitHub repository to Vercel
2. Push to the main branch
3. Vercel automatically deploys your changes
