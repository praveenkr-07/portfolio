## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/praveenkr-07/portfolio.git
```

Move into the project

```bash
cd portfolio
```

Open the project

Simply open **index.html** in your browser or use **Live Server** in VS Code.
